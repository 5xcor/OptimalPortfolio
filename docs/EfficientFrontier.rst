##################
Efficient Frontier
##################

Once the moments of the distribution of market invariants has been estimated,
portfolio weights must be calculated. These weights must optimise certain
utility functions, that is chosen by the user. The classical utility functions
include the Sharpe Ratio, efficient risk, return and minimum volatility. All of
these have been implemented here.
